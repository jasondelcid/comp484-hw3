# Comp 484 - Homework 3 - Links

This assignment focuses on using HTML links to create navigation, external links,
and fragment identifiers within a multi-page site.

## Features Implemented
- Consistent page navigation using document-relative links
- External links that open in a new tab (CSUN website and Twitter)
- Fragment identifiers linking to specific sections within and across pages
- Semantic HTML structure using header, nav, main, section, article, and footer elements

## Files
- `index.html` – Introduction to HTML
- `syntax.html` – HTML syntax overview
- `structuring.html` – Structuring pages with semantic elements
- `links.html` – Internal and external link examples
- `reference.html` – HTML reference and named character entities table
- `next.html` – Next steps and learning resources
